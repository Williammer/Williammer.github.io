## Setup [(refs)](https://help.github.com/articles/setting-up-your-github-pages-site-locally-with-jekyll/)
1. Install ruby gem


2. Install jekyll
``` bash
$ gem install jekyll bundler
```

3. Install github-page
``` bash
$ bundle install
```

4. Run jekyll server
``` bash
$ jekyll s
```
``` bash
$ bundle exec jekyll serve
```

* Keep Github page up-to-date
``` bash
$ bundle update
```
